Helpful Features adds features to WordPress.

== Installation ==

1. Upload the plugin files to the '/wp-content/plugins/' directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Appearance->Customize->Helpful Features screen to configure the plugin
1. Use the Tools->Helpful Features screen to learn more about the plugin
